# TC_/IP Model

The model has four layers

## Application

Examples inlclude:

- The world wide web
- Email
- VOID
- SMS for messages
- IRC

## Transport

Examples include:

- TCP
- UDP

## Internet

Examples include:

- ID address

## Link (Network Access)

Protocols in this layer include:

- Wifi
- Bluetooth
- LTE
- Ethernet
