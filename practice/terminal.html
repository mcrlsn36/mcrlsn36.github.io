This is a guide to the terminal

Features Include:

  • making new directories (mkdir)
  • copy and moving files around (cp and mv)
  • using the command line
  • viewing contents of large files (less)

This is how to make a change in git through the terminal

  1. Type in git status and press enter
  2. Type in git add "filename" and press enter
  3. Type in git commit -m "made a change to index" and press enter
  4. Type in git push and then press enter

After, make sure to refresh git and it should show up.

A Bonus Feature of the Terminal
